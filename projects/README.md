# Projects

Active undertakings, grouped by area.

Path: `projects/<area>/<kebab-title>.md`
For client work, the convention is `projects/<area>/<client-slug>-<project-slug>.md`.

Frontmatter and body skeleton: see `docs/templates/project.md`. Format reference: `CLAUDE.md → Frontmatter conventions → Project`.

A project page should always have a `next-step` field — the single concrete action that moves the project forward. The assistant will use that when you ask "what should I work on next?".
