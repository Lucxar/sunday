# Journal

One file per day, named `YYYY-MM-DD.md`.

Frontmatter and body skeleton: see `docs/templates/journal.md`. Format reference: `CLAUDE.md → Frontmatter conventions → Journal`.

A journal entry typically contains: what happened today, what you learned, what's on for tomorrow, and crosslinks to projects / ideas / topics you touched. The assistant can also write journal entries for you if you describe a day verbally — drop the description in `inbox/`, say "triage", confirm "this is a journal entry for today".
