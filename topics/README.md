# Topics

Knowledge / research nodes. Every concept that more than one source touches deserves its own topic page.

Path:
- Small topic → `topics/<kebab-topic>.md`
- Larger topic with sub-pages → `topics/<kebab-topic>/index.md` + sibling files

Frontmatter and body skeleton: see `docs/templates/topic.md`. Format reference: `CLAUDE.md → Frontmatter conventions → Topic`.

Topics are where the wiki compounds. Each source you ingest updates the relevant topic pages — adding new findings, flagging contradictions with prior claims, bumping `sources-count` and `last-updated`. Over time, the topic page becomes the **synthesis** across everything you've read on that subject. You read the topic; you only dive into raw sources when you need verbatim quotes.
