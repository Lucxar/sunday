# Ideas

One file per idea. Grouped into subfolders by area.

Path: `ideas/<area>/<kebab-title>.md`

Frontmatter and body skeleton: see `docs/templates/idea.md`. Format reference: `CLAUDE.md → Frontmatter conventions → Idea`.

Ideas can graduate to projects (move the file to `projects/<area>/`, change `typ: idea` → `typ: project`, update wikilinks). They can also stay as ideas forever — that's fine, the lint pass will not nag you about it.
